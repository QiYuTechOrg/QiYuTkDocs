WebHook 配置
====================================

通过配置 WebHook 您可以实时获取淘客系统服务器端的一些状态。


当前支持:

* 新订单通知
* 新用户通知
* 绑定渠道ID通知

使用管理员登录后台，在 :code:`constance` 中配置。

.. note::

    通过 `推送系统 <https://notify.qiyutech.tech/>`_ 您可以从微信上实时接收信息。
