奇遇淘客 Android APP#

  • 构建您自己的APK

奇遇淘客 Android APP 是由 奇遇科技 [0] 开发的一款应用于 Android [1] 操作系统的开源淘宝客 APP, 您可以依据 AGPL 3.0 [3] 开源协议自由 & 免费的获取所有的源代码。

获取代码#

您可以从 https://github.com/QiYuTechOrg/QiYuTkAndroid 下载源代码。

构建奇遇淘客 Android APP#

奇遇淘客 APP 使用 Android Studio (当前使用 4.2 版本) [2] 构建, 允许在 Windows、macOS 以及 Ubuntu 操作系统上执行构建过程。

本人使用的 Android Studio 版本信息如下:

macOS 10.16

Cores: 8
Registry: external.system.auto.import.disabled=true

备注

本项目已经在 Github 上部署了持续集成(CI)， 如果您本地无法正常编译，请先检查您本地机器的环境。
