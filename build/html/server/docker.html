Docker 镜像#

为了方便部署，我们使用了 Github Action 的自动集成自动发布 Docker 镜像包。

备注

如果您需要部署奇遇淘客服务器端，请参考 服务器端部署 。

这里的镜像仅供高级用户使用。

警告

如果您在线上部署，请不要使用 latest 版本的 docker 镜像。

Docker Hub 官方镜像#

奇遇淘客 Docker 官方镜像地址

奇遇淘客 Docker Tags 信息 [0] 。

docker pull qiyutech/tbk:${TAG}

警告

${TAG} 应该使用 Docker Tag [0] 列表中的 tag 值替换

阿里云 Docker 镜像#

阿里云镜像地址:

docker pull registry.cn-hangzhou.aliyuncs.com/qiyutech/tbk:${TAG}

警告

${TAG} 应该使用 Docker Tag [0] 列表中的 tag 值替换

备注

中国的用户使用阿里云 Docker 镜像地址速度会比较快

ghcr 镜像#

ghcr 镜像地址:

docker pull ghcr.io/qiyutechdev/tbk:${TAG}

警告

${TAG} 应该使用 docker tag 列表中的 tag 值替换
